Objectives

  • Developing a basic understanding of AJAJ/PHP RESTful DB web services programming.
  • Build a web service that supports CRUD.
  • Use AJAJ, PHP and a MySQL database.
  • Create a functional API with testers.
  • Enable manipulation of data in the guitar shop DB
  • Create a WCF Service Application that implements a simple calculator.
  • Run the service locally and test it.
  • Create a Console Application that consumes the WCF service.
  • Test the console client by calling the calculator operations.
  • Manual simulation testing: Use ModbusPal to manually simulate sensors and verify system behavior against expected outcomes.
  • Use Socket.IO for real-time communication, Send events from client to server, Broadcast data to all connected clients, Update the web page dynamically, and Work collaboratively as a team
